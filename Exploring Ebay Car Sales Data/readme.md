# Limpeza e Análise de Carros Usados do Ebay Alemão
Este repositório tem o objetivo de mostrar um mini projeto onde utilizei Pandas para
analizar e limpar um dataset de carros usados de uma seção do Ebay da Alemanha. Também fiz
alguns gráficos simples com Matplotlib, mas como até aqui ainda não tinha me dedicado a estudar
a lib, fiz apenas algumas demonstrações bem simples.

---
This repository aims to show a mini project in which I used Pandas to analyze and clean
a dataset of used cars from German's eBay. I also did some basic graphs with Matplotlib, but
since I didn't have studied the Library at this moment, It's not the aim of this mini project
to show my skills with it.
